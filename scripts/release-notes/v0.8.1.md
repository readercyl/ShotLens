## 更新内容

- 修复 API 测试偶发误报不可用：测试现在使用真实微型翻译链路，和正式翻译一样支持格式修复与兜底解析。
- 修复第一次输入自定义 API 后立刻测试可能读到旧草稿的问题。
- 修复升级后屏幕录制权限反复失效的问题：GitHub Release 包默认使用本机稳定自签名代码签名，不再使用 ad-hoc 临时签名。

## 注意事项

- 本版本仍跳过 Apple Developer ID 认证和公证，但会使用本机 `ShotLens Local Signing` 证书进行稳定签名。
- 从旧 ad-hoc 版本第一次升级到稳定签名版本时，macOS 可能仍需要重新授权一次屏幕录制；之后继续使用同一签名证书发布，权限不应每次升级都丢失。
